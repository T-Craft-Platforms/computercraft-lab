CC Browser

This browser supports a practical subset of HTML and CSS in ComputerCraft.

Features

  • URL bar
  • Back / forward history
  • Reload/abort button while loading
  • Tabs with page titles
  • Tab close buttons
  • Top-left browser close button
  • Drag-and-drop tab ordering
  • Double-click a tab to show its full title
  • Caret mode (F7) for page text selection
  • F7 indicator in the URL status area when active
  • Mouse wheel and keyboard scrolling
  • Clickable links

Supported HTML

headings, paragraphs, div/span, lists, links, br, hr, pre/code, and img alt placeholders.

Supported CSS (subset)

element/class/id selectors, color/background-color, display, white-space, margin, padding-left, font-weight, text-transform.

Keys

  • Enter in URL bar: navigate
  • Mouse wheel / Up / Down / PageUp / PageDown: scroll
  • Ctrl+L: focus URL bar
  • Ctrl+Left and Ctrl+Right: back and forward
  • F5 or Ctrl+R: reload
  • F7: toggle caret mode
  • In caret mode, arrows/home/end/page keys move text selection
  • Ctrl+A/C/X/V: select all / copy / cut / paste
  • Ctrl+T: new tab
  • Ctrl+W: close tab
  • Ctrl+Tab: next tab
  • Esc: abort load (while loading) or quit
  • Ctrl+Q: quit
