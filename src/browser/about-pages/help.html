CC Browser

This browser supports a practical subset of HTML and CSS in ComputerCraft.

Features

  • URL bar
  • Back / forward history
  • Reload/abort button while loading
  • Tabs with page titles
  • Tab close buttons
  • Top-left browser close button
  • Drag-and-drop tab ordering
  • Double-click a tab to expand it to full tab-bar width (click again to restore)
  • Caret mode (F7) for page text selection
  • F7 indicator in the URL status area when active
  • Vertical scrollbar with track/thumb on scrollable pages
  • Home page at about:home
  • Settings page at about:settings (open from the = menu)
  • Favorites page at about:favorites (open from the = menu)
  • History page at about:history (open from the = menu)
  • Mouse wheel and keyboard scrolling
  • Clickable links

Supported HTML

headings, paragraphs, div/span, lists, links, br, hr, pre/code, img alt placeholders, and forms (form, input major types, select/option, textarea, button).

About page templates support dynamic tokens with refresh intervals, e.g. {{MINECRAFT_TIME@500}}.

Supported CSS (subset)

element/class/id selectors, basic attribute selectors (including [name] and [name=value]), color/background-color, display, white-space, margin, padding-left, font-weight, text-transform, overflow/overflow-x/overflow-y.

Keys

  • Enter in URL bar: navigate
  • Mouse wheel / Up / Down / PageUp / PageDown: scroll
  • Ctrl+L: focus URL bar
  • Ctrl+Left and Ctrl+Right: back and forward
  • F5 or Ctrl+R: reload
  • F7: toggle caret mode
  • In caret mode, arrows/home/end/page keys move text selection
  • Ctrl+A/C/X/V: select all / copy / cut / paste
  • Inside forms: Enter submits, Space toggles checkbox/radio/buttons, Tab/Shift+Tab moves focus
  • Form submit uses background fetch and does not navigate away
  • Ctrl+T: new tab
  • Ctrl+W: close tab
  • Ctrl+Tab: next tab
  • Esc: abort load (while loading) or quit
  • Ctrl+Q: quit
